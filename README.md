# Ukuvota

Ukuvota is a full stack application dedicated for score voting with a cooperative process for organizations and groups.

## Usage
 
```
git clone https://codeberg.org/ukuvota/ukuvota.git
cd ukuvota
pnpm install
pnpm dev
```
